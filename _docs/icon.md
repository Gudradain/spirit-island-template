# Icons

You can use icons in your text to represent the game elements. Here is the syntax:
```
<icon class="class-name"></icon>
```
Then, you just need to replace 'class-name' by the icon name. The available choices are below.

### Invader Icons

- explorer
- town
- city

### Spirit Icons

- presence
- sacred-site
- spirit
